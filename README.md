## 🧠  A Quick Guide: How to Hack Japanese

Before you touch a textbook, you need to understand the three pillars of efficiency for this specific language:

### 1. The SOV Deconstruction
English uses SVO: I (subject) eat (verb) the apple (object).
**The Trick:** Japanese places the action verb at the end of the sentence; everything else in the sentence is just a package of information labeled with a particle. Particles ($wa$, $ga$, $o$, $ni$) are the glue that shows you how to label words in a sentence. If you know the particles, it doesn’t matter where you place your words.

### 2. Radical-Based Kanji (The LEGO Method)
Don't memorize 2,000 individual Kanji. Memorize the **214 Radicals**. 
* **Example:** The Kanji for "Rest" (休) is just "Person" (人) + "Tree" (木). 
* **The Strategy:** Use mnemonics. A person leaning against a tree is *resting*. By learning the building blocks, you decode new characters on sight instead of rote memorization.



### 3. The Pareto Principle (80/20 Rule)
60% of daily Japanese conversation consists of just 500-800 words. We will prioritize the **Core 2k Deck** (the 2,000 most common words) via Anki to ensure you aren't wasting time on "encyclopedia" or "astronomy" terms in Week 3.

### 2. DAILY STUDY TEMPLATE (WEEKDAY)

**Repeatable 75-minute routine (Mon–Sat). Total cap: 90 min max. Adapt new cards to weekly focus (always pull from top-frequency list + current grammar).**

- **0–5 min: Preparation & Meta-Setup**  
  Review weekly focus topics and KPIs. Set Anki new-card limit. 30-second pitch-accent warm-up (listen to 5 minimal pairs and repeat).

- **5–30 min: SRS Anki Session (25 min)**  
  Complete ALL due reviews first. Then add exactly 20 new cards (15 vocabulary from frequency list + 5 grammar/sentence cards). Use “Again/Hard/Good/Easy” honestly. If reviews exceed 35 min total, drop new cards to 15 tomorrow.

- **30–45 min: Grammar Deconstruction & Pattern Practice (15 min)**  
  Study 2–3 new grammar points (or review weak ones). Break down using meta-learning (see section 5). Create and write/say 5 original example sentences.

- **45–65 min: Comprehensible Input (20 min)**  
  10 min graded reading (i+1 material — look up max 2 words).  
  10 min listening + shadowing (repeat after native speaker, match pitch exactly).

- **65–75 min: Active Output & Reflection (10 min)**  
  Record yourself saying 3–5 new sentences. Self-correct pronunciation/pitch.  
  2-minute reflection: note 1 success + 1 error. Switch to “Japanese mode” (mentally label 3 room objects in Japanese).

**Total: 75 minutes.** If under 60 min, add 5 min extra shadowing. No romaji after Week 2.

### 3. SUNDAY REVIEW SYSTEM

**Weekly Review Checklist (60–90 min total, active recall only — no new learning)**  
- [ ] Finish every overdue Anki card (target 100% correct on this week’s new cards).  
- [ ] Re-read/re-listen to all week’s graded readers and podcasts **without** transcripts or lookups.  
- [ ] Active recall test: Write 20 original sentences using only this week’s grammar + vocab (cover answers, check accuracy).  
- [ ] Listening self-test: Play week’s input at normal speed → summarize key points in Japanese (simple sentences) or English.  
- [ ] Random quiz: 50 cards via Anki custom study (or paper flashcards) on vocab + kanji.  
- [ ] Error journal: List 5 mistakes and fix them with 1 new example sentence each.

**Active Recall Strategies**  
- Cloze deletion on full sentences in Anki.  
- “Teach-back”: Explain each grammar point aloud as if teaching a friend.  
- Cover-and-recall: Hide English side, produce Japanese from meaning.

**Self-Testing Methods**  
- Record 2-min monologue using week’s patterns → compare to native audio.  
- Speed reading test: Time yourself reading 1 page of graded reader (goal: <90 seconds with 90% comprehension).

**Weak-Point Diagnosis Process**  
1. Open Anki stats → filter leech cards (lapsed >2 times) and cards with interval <3 days.  
2. Categorize errors (particle mix-up, pitch, vocab interference, grammar form).  
3. Decide fix: +5 extra sentence cards on weak pattern OR +10 min targeted input next week.  
4. Update next week’s focus if any KPI was missed.

### 4. ANKI SYSTEM DESIGN

**Card Formats (create all cards with context — never isolated words)**

1. **Vocabulary Cards**  
   - Front: Japanese (kana/kanji) + audio (record or use Forvo)  
   - Back: English meaning + full example sentence (Japanese + English translation) + pitch accent notation (e.g., たべる [HLL])  
   Example:  
   Front: 食べる  
   Back: to eat  
   私は毎日ご飯を食べます。 (I eat rice every day.)

2. **Grammar / Sentence Cards** (cloze or production)  
   - Front: Sentence with target grammar highlighted or blanked (e.g., 毎日___食べます。)  
   - Back: Full rule + explanation + 2 additional example sentences  
   Example:  
   Front: 私は毎日ご飯を___ます。 (polite present)  
   Back: 食べます → Rule: verb stem + ます (polite present). Used for daily routines.

3. **Kanji Cards**  
   - Front: Kanji only  
   - Back: On/kun readings, English meaning, 2 example words + sentences

**Daily Limits**  
- New cards: 20 maximum (15 vocab + 5 grammar/kanji). Reduce to 15 if reviews take >35 min.  
- Reviews: All due cards (no limit, but finish in one sitting). Aim for <40 min total review time.

**Optimization Tips**  
- Enable “bury related cards” and “leech threshold = 8 lapses”.  
- Always mine sentences directly from weekly input sources (after Week 4).  
- Use FSFS (Anki’s built-in spaced repetition) or enable FSRS addon for smarter scheduling.  
- Add audio to every card (use AwesomeTTS or record yourself).  
- Weekly: Export stats and reset new-card limit Sunday night based on review burden.

### 5. META-LEARNING PLAYBOOK

**How to break down Japanese grammar efficiently (5-minute process)**  
1. Identify structure: Topic (は) → Comment → Verb (end of sentence).  
2. Isolate particle function (e.g., は = “as for X”; が = new/contrast information).  
3. Strip verb to stem + ending (食べ + ます = polite present).  
4. Map to English difference: Japanese is SOV + context-first; English is SVO + explicit subjects.  
5. Create 3 original sentences immediately.

**Mental models for sentence structure**  
- Japanese = **Topic + Comment** (not Subject-Verb-Object).  
- Particles replace English word order/prepositions.  
- Context is king: omit what is obvious (no “I” if clear from topic).  
- Pitch accent changes meaning — treat it like a 5th “particle”.

**Fast pattern recognition strategies**  
- Chunking: Learn “～て ください” as single unit = “please do ~”.  
- Pattern spotting: After 50+ sentences, predict particle before reading (e.g., after 私, expect は or が).  
- Minimal-pair drilling: Compare 私は vs 私が in 10 sentences daily until automatic.  
- Input-first: Read/listen 10x more than you output — patterns emerge naturally.

**Common beginner mistakes + fixes**  
- **Romaji dependence** → Fix: Delete all romaji after Week 2; force kana-only Anki and input.  
- **Wa/ga confusion** → Fix: 20 minimal-pair sentences daily + input exposure only (no translation).  
- **Flat pitch accent** → Fix: Shadow every new card; use Forvo/OJAD reference before marking “Good”.  
- **Literal English translation** → Fix: Daily “Japanese mode” block — think only in Japanese for 5 min; label room items in Japanese.  
- **Interference with other languages** → Fix: Separate study environment (Japanese-only browser tab, phone language switch during session, no mixing notes).
